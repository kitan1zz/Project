#### `HeroSection.tsx`
- Hero-секция с анимированным заголовком
- Градиентный текст
- Декоративные анимированные элементы фона
- Поддержка темной/светлой темы

#### `SectionCard.tsx`
- Переиспользуемая карточка для разделов
- Анимация при наведении
- Поддержка иконок и цветов
- Плавные переходы

#### `FadeInOnScroll.tsx`
- Компонент для анимации при скролле
- Использует Intersection Observer
- Настраиваемые направления анимации
- Появляется один раз при входе в viewport

#### `InfoCard.tsx`
- Информационная карточка с иконкой
- Варианты: info, warning, success
- Адаптивный дизайн
- Поддержка темной темы
